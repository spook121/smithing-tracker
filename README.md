Smithing Tracker is a RuneLite plugin that tracks everything while you smith.

What it tracks:

Status - Shows if you are Idle, Banking, Running, or Smithing (Anvil) Cannonball smithing included.
Current Item - Detects what you are smithing
Profit/Session Stats - Items smithed per hour, runtime, Profit/hr aswell as total profit made(i.e. if you buy 10k runite bars and craft 1 rune 2h, it will show the data for the profit made for those r2h smithed)

SMITHED ITEMS Box:

Shows every item you have in Bank + Inventory that is smithable
Each item gets its own icon
Yellow number under each icon = total count you own (bank + inv)
Only shows items you enable in filters
Filters - Per Metal:
Separate toggles for Bronze, Iron, Steel, Mithril, Adamant, Rune. Each section has:

ALL - Select/Deselect All - one click to check/uncheck every item in that metal
Individual checkboxes for all 27 smithing items: Dagger, Axe, Mace, Med Helm, Bolts, Sword, Nails, Dart Tips, Scimitar, Spear, Hasta, Arrowtips, Limbs, Longsword, Javelin Tips, Full Helm, Knives, Sq Shield, Warhammer, Battleaxe, Chainbody, Kiteshield, Claws, 2h Sword, Platelegs, Plateskirt, Platebody, plus Cannonball
Pickaxes are fully excluded (they were showing as Axes before)
